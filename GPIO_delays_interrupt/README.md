# GPIO, Delays, Interrupt

My first project interfacing with the ATMega328PB, with the purpose of practicing the use of timers, interrupt subroutines, and I/O pins (to led).

[Video demonstration](https://youtu.be/kt5ZJVvbXw0)

Task 1 was to light for 1.5 sec via polling.\
Task 2 was to light for 3 sec via interrupt.\
Task 3 was both combined on separate buttons.

All were completed in both assembly and C code.
