# Multi-functional Shield PCB Design

For this project, I was tasked with taking all of my previous ATMega328PB projects and fitting all of the components necessary for them to function to an Arduino compatible multi-functional shield. The whole process is detailed in [the report](https://github.com/CollateralJ/embedded-projects/blob/main/PCB_multifunctional_shield_design.pdf).

\
This PCB would add LEDs, push buttons, a potentiometer, a 7-segment display, headers for motor connections, an OLED display, and a BMI160 inertial measurement unit for plug and play use with my embedded projects!
